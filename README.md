# Personal Finance Tracker (Console App)

Консольное приложение для ведения личных финансов.
Позволяет учитывать доходы и расходы, просматривать историю операций и рассчитывать текущий баланс.

Проект создан в учебных целях для практики базового C# и принципов объектно-ориентированного программирования.

# 📌 Функциональность

Добавление доходов
Добавление расходов с проверкой на недостаток средств
Просмотр всей истории операций
Расчёт и отображение текущего баланса
Очистка истории операций

# Цветной вывод сообщений в консоль:

доходы — зелёный
расходы — красный
ошибки — красный
успешные операции — зелёный

# 🧱 Архитектура проекта

Проект разделён на логические слои и пространства имён:

# Основные классы
# Transaction
Хранит данные одной финансовой операции:
тип операции (доход / расход)
описание
сумма
Переопределён метод ToString() для удобного вывода в консоль.

# TransactionHistory

Отвечает за:
хранение списка транзакций
добавление операций
расчёт баланса
очистку истории
Является единым объектом на всё приложение.

# TransactionInput

Отвечает за ввод данных пользователем:
ввод и валидацию описания транзакции
ввод и валидацию суммы дохода
ввод суммы расхода
Использует циклы while для повторного запроса данных при ошибочном вводе.

# Menu

Отвечает за отображение текстового меню в консоли.

# ConsoleOutput

Содержит методы для форматированного вывода сообщений:
успешные операции
ошибки
отсутствие транзакций
Использует изменение цвета текста консоли.

# Program

Точка входа приложения.
Содержит:
основной цикл while
обработку пользовательского выбора
взаимодействие между компонентами системы

# 🔁 Логика работы приложения

Пользователю отображается меню
Пользователь выбирает действие

Приложение:
запрашивает данные
валидирует ввод
выполняет операцию
После выполнения возвращается в главное меню
Работа завершается при выборе пункта «Выход»

# ⚙️ Используемые технологии

switch-case

Работа с Console
